# 📊 Restaurant Sales & Performance Dashboard

Progetto di Business Intelligence e analisi delle vendite sviluppato in **Power BI**.

## 🎯 Obiettivo del Progetto
Analizzare le performance commerciali, i volumi di vendita e il comportamento d'acquisto dei clienti per ottimizzare la gestione dell'attività di ristorazione.

## 🛠️ Strumenti Utilizzati
* **Power Query:** Pulizia, trasformazione e modellazione dei dati dal file CSV di origine.
* **Power BI:** Modellazione dati e Data Visualization.
* **DAX:** Creazione di misure calcolate (Totale Ordini, Fatturato, Scontrino Medio).

## 📈 Key Insights
* **Volumi:** Gestione di **15.776 ordini** totali per un fatturato complessivo di **$3,15M**.
* **Scontrino Medio:** **$199,47**.
* **Categorie Top:** La categoria *Main Dishes* guida i ricavi generando il **47,08%** del fatturato totale.
* **Prodotti:** Individuazione immediata dei top performer (es. *Grilled Chicken*) e dei prodotti meno venduti (es. *Iced Tea*).
